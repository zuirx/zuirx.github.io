Baixando coisas do Youtube

Isso é algo que a empresa está contantemente tentando impedir, então o que posso colocar aqui pode parar de funcionar amanhã ou não.

Via web (mais fácil e rápido)

Site:cobalt

Apenas cola o link na barra, selecione "auto", e enter.

Via celular (Android)

App:NewPipe

Para quem lembra SnapTube, isso pode ser uma boa alternativa, mas saiba que ele pode parar de funcionar se o Google quiser..

Via linha de comando

Pode ser mais complicado, mas é mais confiável e sempre atualizado.

Para instalar no Windows, abra Powershell (sem administrador), e cole as linhas abaixo:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
scoop install yt-dlp

Após instalar, execute a linha para baixar de um link:

yt-dlp -f [CÓDIGO] "[LINK AQUI]"

Em [CÓDIGO], pode variar se o Youtube mudar, mas geralmente é o número 18 para vídeo e 140 para música. Essa é a funcionalidade básica dele, pesquisando o nome do programa mostra mais jeitos de como usá-lo.
