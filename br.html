Fazendo um RubberDucky USB simples em casa

05-09-2024 tutorial

RubberDucky oficial é legal, mas caro. Há uma solução alternativa...

Você pode comprar um desses chips a um custo muito baixo (~5$).

Depois, siga os passos de instalação:

1 - Instale os drivers em github.

1.1 – Extraia o arquivo, abra DPinst64.exe, clique em avançar várias vezes.

2 - Instale o Arduino IDE (versão antiga) do site oficial.

3 - Abra o Arduino, vá em Arquivo > Preferências, e na opção "URLs Adicionais de Gerenciadores de Placas", cole:

https://raw.githubusercontent.com/digistump/arduino-boards-index/refs/heads/master/package_digistump_index.json

e clique em OK.

3.1 - Vá em Ferramentas, na opção Placa: [alguma placa aqui se estiver conectada], clique em Gerenciador de Placas.

3.2 - Procure por "Digitump", instale "Digitump AVR Boards", e feche após instalar.

4 - Com tudo instalado, programe seu dispositivo, use o código abaixo como exemplo (este abrirá a calculadora no Windows):

#include "DigiKeyboard.h"
void setup() { }

void loop() {
DigiKeyboard.sendKeyStroke(0);
DigiKeyboard.delay(500);
DigiKeyboard.sendKeyStroke(KEY_R, MOD_GUI_LEFT);
DigiKeyboard.delay(500);
DigiKeyboard.print("calc");
DigiKeyboard.sendKeyStroke(KEY_ENTER);
DigiKeyboard.delay(500);

for (;;) {
/*loop vazio*/
}

}

5 - Para enviar, desconecte a placa, pressione Ctrl+U, no console reconecte a placa USB se solicitado, e o script será executado.

6 - Confira estes scripts também.

E o resto é história.

Melhor maneira de baixar conteúdo do YouTube

03-09-2024 tutorial

Isso é algo que o Google está constantemente tentando impedir de funcionar, então o que coloco aqui pode parar de funcionar amanhã ou não.

E você não pode fazer isso no YouTube, é claro, então ficará aqui.

Maneira fácil

Cobalt - basta colar o link, selecionar automático para vídeo ou áudio, e pressionar Enter.

Maneira no Android

NewPipe - instale o app, selecione o vídeo e clique em baixar. Funciona bem como alternativa ao aplicativo do YouTube, mas pode ser instável.

Linha de comando (mais confiável)

Mais difícil, mas eficiente.

Uma maneira fácil de instalar no Windows: abra o cmd/powershell sem admin e execute abaixo:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
scoop install yt-dlp

Após instalar, obtenha os formatos de conteúdo disponíveis para download:

yt-dlp -F "[LINK]"

Geralmente 18 é um vídeo com áudio em qualidade regular e 140 para áudio apenas. Após encontrar o melhor formato, execute: yt-dlp -f 18 "[LINK]"

Verifique sua documentação para mais comandos.
